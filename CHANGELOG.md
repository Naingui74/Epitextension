# Changelog

Toutes les modifications notables de ce projet seront documentées dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère au [Versioning Sémantique](https://semver.org/lang/fr/).

## [2.2.0] - 2024-01-XX

### Ajouté
- Support de la nouvelle URL `myresults.epitest.eu`
- Compatibilité avec l'ancienne URL `my.epitech.eu` (rétrocompatibilité)
- Interface popup complète avec statistiques en temps réel
- Support des raccourcis clavier pour toutes les fonctionnalités
- Gestion des paramètres avec sauvegarde persistante
- Mode compact pour la vue des différences de tests
- Statistiques avancées (tests analysés, projets suivis)
- Service Worker pour de meilleures performances
- Gestion d'erreurs robuste avec fallbacks
- Scripts de build automatisés
- Configuration ESLint pour la qualité du code
- Support complet Manifest v3

### Modifié
- Mise à jour du manifest vers la version 3
- Amélioration de la gestion des thèmes (automatique, manuel)
- Optimisation des performances avec debouncing
- Amélioration de l'interface utilisateur
- Refactorisation du code pour une meilleure maintenabilité

### Corrigé
- Gestion des erreurs lors de la copie dans le presse-papiers
- Stabilité de l'interface popup
- Synchronisation des paramètres entre les onglets
- Gestion des erreurs réseau

### Technique
- Migration vers Manifest v3
- Ajout d'un Service Worker
- Configuration ESLint
- Scripts de build automatisés
- Amélioration de la structure du code
